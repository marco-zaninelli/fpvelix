# Fpvelix

Fpvelix is a portfolio website for a German drone videographer designed to improve brand awareness and showcase completed work.

## Features

- Responsive design for desktops, tablets, and mobile devices
- Multilingual support (German and English)
- Portfolio showcase with video content
- Integrated CMS using Sanity Studio for easy content management
- Built with modern technologies for fast performance and scalability

## Technology Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Sanity Studio](https://www.sanity.io/) for content management
- Multilingual support implemented for smooth internationalization
